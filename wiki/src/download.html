[[!meta title="Download Tails"]]

Every ISO image we ship is a hybrid one:

  • it can be burnt to a CD;
  • it also contains a full disk image (including partition table) and can thus be [[installed onto a USB stick|support/Installing onto a USB stick]]; the USB stick's content is lost in the operation.

Any help to spread Tails is more than welcome, [[using BitTorrent or by setting up a web mirror|contribute/how/mirror]].

If you are interested in the source code, have a look to our [[dev zone|contribute]].

2. Verify the downloaded image integrity

The released images are signed with our [[GnuPG key]]; you can, and should, verify their integrity before using them.

Users of a recent enough GNOME should make sure they have the seahorse-plugins package installed, then right-click on the .asc OpenPGP signature file, and open it with the Verify signature tool.

Others rather need to do this on the command-line, in the directory where the .iso and .asc files are: 

gpg --verify tails*.iso.asc

3. Install the system

You now may:

  • burn the ISO image to a CD;
  • or install the ISO image onto a USB stick: just follow [[our dedicated how-to|support/Installing onto a USB stick]].
