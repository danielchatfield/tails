[[!meta title="Download T(A)ILS"]]

Every ISO image we ship is a hybrid one:

  • it can be burnt to a CD;
  • it also contains a full disk image (including partition table) and can thus be copied (using `dd`) to a USB stick; the USB stick's content is lost in the operation.

Any help to spread T(A)ILS is more than welcome: don't hesitate seeding our Torrents. If you want to setup a web mirror, please [[drop us an email|contribute]]. FIXME: link to the related documentation page.

2. Verify the downloaded image integrity

The released images are signed with our [[GnuPG key]]; you can, and should, verify their integrity before using them. Example: 
gpg --verify amnesia*.iso.asc
