[[!meta title="Upgrade"]]

Upgrading Tails is just like installing it for the first time:

  • if you are using Tails on a DVD, just burn a new one; time to get a DVD-RW to ease the process?
  • if you are using Tails on a USB stick, just install the latest version on it; beware: any personal data stored on the stick will be lost!

The [[download]] page explains how to proceed.
